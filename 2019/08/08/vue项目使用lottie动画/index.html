    以上配置只用到了animationData,下面列出一些常用的，更多配置项请在该链接中查看

    • animationData: 动画json
    • path: 可直接指定动画json // 路径
    • loop: true / false / number //循环

    • name: // 给动画取名字,方便方法调用
    • renderer: ‘svg’ / ‘canvas’ / ‘html’ //渲染方式
